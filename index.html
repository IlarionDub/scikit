sklearn.linear_model.LogisticRegression

class LogisticRegression(penalty='l2', *, dual=False, tol=0.0001, C=1.0, fit_intercept=True, max_iter=100, solver='lbfgs', n_jobs=None)

LogisticRegression — лінійний класифікатор, який моделює ймовірність належності об’єкта до класу за допомогою логістичної функції. Підтримує L1/L2 регуляризацію, різні солвери, багатокласову класифікацію та роботу з вагами класів.

Parameters

C : float, default=1.0 Обернений до сили регуляризації. Менші значення означають сильнішу регуляризацію.
penalty : {'l1','l2','none'} Тип регуляризації, який застосовується до коефіцієнтів моделі.
solver : str Алгоритм оптимізації (наприклад, 'lbfgs', 'liblinear'). Деякі солвери підтримують не всі типи регуляризації.

Attributes

  • coef_ — ваги ознак після навчання моделі.
  • intercept_ — вільний член (зсув).
  • n_iter_ — кількість ітерацій, виконаних солвером.

Examples

Простий приклад використання:

from sklearn.linear_model import LogisticRegression
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split

X, y = load_iris(return_X_y=True)
X_train, X_test, y_train, y_test = train_test_split(X, y, random_state=42)

clf = LogisticRegression(max_iter=1000)
clf.fit(X_train, y_train)

print("Accuracy:", clf.score(X_test, y_test))
